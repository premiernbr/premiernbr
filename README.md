### 🌍 نبذة / About

<p dir="rtl" align="right">
<b>العربية (AR)</b><br/>
مطوّر وباحث في المواطنة الرقمية والعدالة المعرفية. أركّز على سؤال بسيط وصعب في آن واحد: كيف يغيّر الذكاء الاصطناعي لغات البرمجة، وطريقة عملنا، وشكل المجتمع؟ أشتغل على حلول تجمع بين الذكاء الاصطناعي السحابي والمحلي، وتقنيات الاسترجاع المعزَّز للتوليد (RAG)، وأدوات الأتمتة لبناء منظومات معرفة مفتوحة وأكثر عدلًا وشفافية.

### 🧠 مجالات الاستكشاف / What I’m exploring

<p dir="rtl" align="right">
أستكشف كيف يمكن للذكاء الاصطناعي أن يساعد الإنسان على التفكير بوضوح وعمق، بدل أن يكون مجرد "شات يجيب على الأسئلة".<br/>

أجرّب دمج البيانات الرسمية والبيانات المفتوحة في مساعدين رقميين موثوقين يعتمدون على الاسترجاع المعزَّز للتوليد (RAG).<br/>

</p>

**Short EN version**  

- AI that helps humans think better, not just “a chat that answers questions”.  
- How AI is changing programming careers, learning paths and daily work.  
- Using RAG to combine official and open data into trustworthy digital assistants.  
- Knowledge justice: equal access to clear information in a language people actually understand.

### 📚 مشاريع وتجارب / Projects & experiments

<p dir="rtl" align="right">
<b>المرشد الرقمي / LegalRAG</b><br/>
مساعد قانوني رقمي موجه للمغرب، يدمج بين النماذج اللغوية (LLMs) وقواعد بيانات قانونية محلية (قوانين، مساطر، دوريات، خرائط قضائية...). الهدف هو بناء واجهة حوارية بسيطة، لكن خلفها منظومة RAG دقيقة تحترم النصوص الرسمية، والخصوصية، وحدود الإفتاء القانوني.
</p>

<p dir="rtl" align="right">
<b>مساعدون رقميون للعدالة (Digital justice assistants)</b><br/>
تجارب في بناء مساعدين رقميين يعتمدون على البيانات الرسمية والمفتوحة، مع تركيز على الشرح المبسّط واحترام الخصوصية وحدود القانون.
</p>

<p dir="rtl" align="right">
<b>الذكاء الاصطناعي ولغات البرمجة (AI & programming languages)</b><br/>
محاولات لاستغلال نماذج اللغة في فهم الكود، وتحسين التوثيق، ومساعدة المبرمجين على التعلّم الذاتي بهدوء وفعالية.
</p>

<p dir="rtl" align="right">
<b>البيانات المفتوحة والخرائط (Open data & maps)</b><br/>
تجميع ومعالجة بيانات مفتوحة لصنع خرائط ومعطيات تساعد على فهم الواقع، وتغذية قرارات أكثر وعيًا في الإدارة والمجتمع.
</p>

*(Some of these experiments are still in internal development; I publish what can be shared gradually.)*

### ⌨️ Daily workflow

    # local + cloud AI workflow
    pwsh ./run.ps1          # prepare env, start services, load RAG pipelines

    # keep repositories clean
    git status              # review changes
    git add .               # stage updates
    git commit -m "chore: refine RAG and prompts"

    # run local AI tools (examples)
    lm-studio &             # start local models for offline experiments
    # ollama run llama3     # alternative local model runner

    # when needed, compare with cloud
    # curl https://api.openai.com/v1/chat/completions ...

<p dir="rtl" align="right">
أستعمل سكريبتات بـ PowerShell لتهيئة البيئة المحلية، وتشغيل الخدمات، وربطها بقنوات الاسترجاع المعزَّز للتوليد (RAG).<br/>
أتابع حالة المستودعات بـ Git وأحافظ على تاريخ نظيف وواضح للتغييرات.<br/>
أجرّب النماذج المحلية عبر أدوات مثل LM Studio (وغيرها)، خصوصًا عندما لا يكون الاتصال السحابي خيارًا مناسبًا.<br/>
وعند الحاجة، أقارن بين أداء النماذج المحلية والخدمات السحابية لاختيار التوازن الأفضل بين الخصوصية والجودة والتكلفة.
</p>

### 🏛️ الصمعة الرقمية / Digital Minaret

مشروع طويل النفس يبني "منارة رقمية" تربط بين:

- هيئة كتابة الضبط، والمحاكم والخدمات القضائية في المغرب.
- بوابات البيانات المفتوحة الرسمية (وزارة العدل، وزارة الداخلية، data.gov.ma…).
- مساعدين رقميين يعتمدون على تقنيات الاسترجاع المعزَّز للتوليد (RAG)، وعلى نماذج لغوية محلية وسحابية.

الهدف البعيد:

> أن يحصل المواطن وموظف العدالة على معلومة واضحة، موثوقة، وباللغة التي يفهمها، مع احترام الخصوصية والقانون وحدود الإفتاء القانوني.
